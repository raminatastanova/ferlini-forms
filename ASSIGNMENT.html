Assignment

Create two views using SwiftUI. The exercise focuses on implementing the required functionality with particular attention on code quality, architecture and less on design.

Create a Form View

You need to create a Form to allow users to create and edit user data. This form should have the following fields:

  1. Name: Text field for entering the user's name.
  2. Country: A picker or a list of countries fetched from the provided API (https://restcountries.com).
  3. Phone Number: Text field for entering the user's phone number.
  4. Email: Text field for entering the user's email.

Validation:

  • Name: Ensure that the name is not empty.
  • Country: Select a country from the list.
  • Phone Number: Validate that it's a valid phone number (you can use simple validation like checking for a valid format).
  • Email: Validate that it's a valid email address (you can use simple validation like checking for "@" and "." presence).

Data Storage:

Save the user's data in a data structure, such as a Plist or Core Data. You should save all the entered information for each user.

Create a User List View

Create a second view that displays a list of users and allows users to perform the following actions:

  1. Load Form View: When a user taps on a user in the list, open the Form View with the selected user's information pre-filled, allowing users to edit the details.
  2. Create New User: Add a button to create a new user. Tapping this button should open the Form View with empty fields.

Remember that the main focus here is code quality, architecture and good design principles.

Please note that you will need to write Swift code using SwiftUI or UIKit to implement these views.
